'macos' 'google-cloud-sdk'

1. install google cloud SDK
Download form 
https://cloud.google.com/sdk/docs/quickstart-mac-os-x
unzip and run installation by
```
$ sh install.sh
```
2. gcloud init
run 
```
$ gcloud init
```
3.
list all instances on your gcloud
```
$ gcloud compute instances list
```
4.
```
$ gcloud 
```
